# Retail Sales & Customer Insights Analysis

## 📌 Project Overview
This project focuses on analyzing retail sales data to understand customer purchasing behavior, category performance, and sales trends.  
The analysis combines Python (data cleaning + EDA) and SQL (business insights extraction) to generate actionable insights.

---

## 📊 Dataset Information
- Size: 10,000+ records  
- Key columns: Branch, City, Category, Unit Price, Quantity, Date, Time, Payment Method, Rating, Profit Margin, Total Sales  

---

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (MySQL)
- Jupyter Notebook / Google Colab

---

## 🔄 Workflow
1. Data cleaning using Python (handling nulls, duplicates, formatting)
2. Feature engineering (Total Sales calculation)
3. Exploratory Data Analysis (EDA)
4. SQL-based business analysis for deeper insights

---

## 📈 Key Insights

### 1. Sales Distribution
- More than **50% of total transactions are below $100**
- Indicates that most customers make low to mid-value purchases

### 2. Outliers in Sales
- Significant outliers observed above **$900+ transaction value**
- These represent bulk purchases or high-value transactions

### 3. Top Performing Categories
- **Fashion Accessories** and **Home & Lifestyle** are the top contributors
- This is based on **total revenue (sum of sales)** per category

### 4. Peak Sales Time
- Highest customer activity observed around **3 PM**
- This is based on **number of transactions (not revenue)**
- Indicates peak shopping engagement during afternoon hours

---

## 📌 Key Learnings
- Data cleaning and preprocessing using Python
- Exploratory Data Analysis (EDA) for understanding data distribution and patterns
- SQL-based business analysis using aggregation and advanced queries
- Converting raw data into meaningful business insights

---

## 🚀 Conclusion
This project demonstrates how retail sales data can be analyzed using Python and SQL to extract meaningful insights about:
- customer behavior  
- product performance  
- time-based trends  

These insights can help businesses improve decision-making and optimize sales strategies.
